Library App

A simple web-based Library App that allows users to add, remove, and manage their book collection. Users can mark books as read/unread and bookmark them for quick access.

Todo List App

A simple and interactive Todo List web application that helps users organize their tasks efficiently. Users can add todos under different topics, set due dates and priorities, mark tasks as completed, edit or delete them, and view completed tasks separately.
